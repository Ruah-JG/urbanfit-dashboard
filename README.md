# UrbanFit Member Engagement Dashboard

## Overview
This project is an interactive data visualization dashboard built using D3.js. It analyzes member behavior, movement patterns between fitness clubs, and class participation trends over time.

The dashboard helps UrbanFit management make data-driven decisions related to operations, marketing, and customer engagement.

---

## Features
- Interactive filters (Month, Membership Tier, Home Club)
- Tooltips and hover interactions
- Dynamic visual updates
- Clean and responsive design

---

## Visualizations

### 1. Member Movement Map
- Displays movement flows between club locations
- Line thickness represents flow volume
- Helps identify key movement hubs

### 2. Parallel Coordinates Plot
- Compares multiple member attributes:
  - Age
  - Monthly Fee
  - Check-ins
  - App Usage
  - Unique Clubs
  - Total Sales
- Colored by membership tier

### 3. Streamgraph
- Shows class participation trends over time
- Highlights top-performing fitness classes
- Helps identify changing user preferences
